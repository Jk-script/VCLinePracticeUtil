# VC_LinePracticeUtil (EN/JP)

Same concept as yuyu's original, rebuilt from scratch with bug fixes and JP 1.0 support.

## Setup

Drop the `.asi` and `.ini` into your VC folder. Needs an ASI loader.

## Keys

```ini
[Keys]
Save=VK_TAB
LoadVehicle=1
LoadOnFoot=2
```

Change them to whatever you want. Supports VK_TAB, VK_F1–VK_F12, or any single 
character like 1, A, G, etc.
